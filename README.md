# 周报迁移 Skill（小组 otl → 部门 ksheet）

将**组内智能文档周报**（`.otl`）中每位成员的内容，自动填入**部门周报表格**（`.ksheet`）对应单元格。  
**你只用在对话里说一句话**；环境配置、脚本执行、成员解析、写回云端均由 Cursor Agent 完成。

---

## 你能做什么 / 不能做什么

| 你只需 | Agent 自动完成 |
|--------|----------------|
| 在对话里触发 Skill（见下文提示词） | 安装依赖、预检环境 |
| 首次提供两个云文档链接 | 读取组内周报、解析成员名单 |
| 缺凭证时粘贴一次 `wps_sid` | 下载部门表、生成计划、写回云端 |
| 预览后点「确认写回」 | 格式化、保留超链接、校验结果 |

| 你不需要 | 原因 |
|----------|------|
| 运行任何 `python` 命令 | Agent 代跑 |
| 提供组成员名单 | 从组内 otl `## 姓名` 自动读取 |
| 未指定周次时选「哪一周」 | 默认组内**最新一期** |
| 手动改 `config.json` | Agent 维护配置 |

## 第一次使用（完整流程）

### 你需要准备

1. **Cursor** 已打开本 Skill 所在工作区（或已安装 Skill）
2. 能登录 **金山文档** 的浏览器（用于获取 `wps_sid`，仅首次或过期时需要）
3. 两个文档链接：
   - **组内周报**（`.otl` 智能文档，每人 `## 姓名` 一节）
   - **部门周报**（`.ksheet` 表格，按人名 + 周次分列）

### 对话里会发生什么

```
你：周报迁移

Agent：正在检查环境…
      （首次）请提供组内周报链接？
你：  https://365.kdocs.cn/l/xxxxxx

Agent：请提供部门周报链接？
你：  https://365.kdocs.cn/l/yyyyyy

Agent：（若缺 wps_sid）请按以下步骤获取 Cookie… 请在下一条消息粘贴 wps_sid
你：  （粘贴一长串字符）

Agent：预览 — 16 人将写入「应用研发-AI应用组」C 列，周次 2026-07-02 …
      是否确认写回云端？
你：  确认

Agent：已写回 v21，请 Ctrl+F5 刷新部门表查看
```

**你不需要打开终端。**

### 首次可能多做的事

| 情况 | 你要做的 |
|------|----------|
| 缺 `wps_sid` | 按 Agent 图文步骤 F12 复制 Cookie，**粘贴到对话** |
| 部门子表名未识别 | 从 Agent 给出的子表列表里选一个名字 |
| 部门表尚未开本周列 | **不用管** — Agent 会按规则插入新列再粘贴 |

## 每周例行使用

组内 otl 写完后，在 Cursor 说一句：

```
周报迁移
```

即可。Agent 会：

1. 取组内**最新一周**内容  
2. 对齐部门表同一周列（没有则插入列）  
3. 预览 → 等你确认 → 写回  

若本周要同步**上周**的：

```
周报迁移，粘贴上一周的
```

## 注意点（重要）

### 1. 文档与权限

- 两个云文档链接需你有**读/写**权限（至少部门表可编辑）
- 组内 otl 须按约定写：`# 日期` 分周，`## 姓名` 分人（见组内模板）
- 部门表为人名一列、周次多列（详见 [references/mapping-rules.md](references/mapping-rules.md)）

### 2. 写回范围（Agent 强制遵守）

- **只改**你指定那一周的**内容列**单元格  
- **不改**历史周列、其它组子表、已有超链接（如 D 列「📄组内周报」）  
- 粘贴内容**只做排版**（列表符、缩进），**不改正文**

### 3. 浏览器查看

- 写回后请 **Ctrl+F5 强刷** 部门表，否则可能看到旧缓存

### 4. `wps_sid` 会过期

- 过几天若迁移失败，按 Agent 提示**重新粘贴**即可，无需重装 Skill

### 5. 不要自己跑脚本

- `scripts/` 下命令仅供 Agent 调用；用户无需也不应手动执行

### 6. 本地缓存会自动清理

- 每次**写回成功**后，Agent 默认清空 `.cache/` 里的中间文件（下载的表格、提取结果等）
- 若你要留档排查，在对话里说：**「保留缓存」** 或 **「不要清理 .cache」**

## 常见问题

| 现象 | 处理 |
|------|------|
| Agent 让你运行 `python …` | 回复「请按 Skill 要求你自动执行」 |
| 预览人数少于组内人数 | 组内 otl 该周该人未写或标题不匹配；补 otl 或让 Agent 跳过 |
| 粘贴后链接能点、换行正常 | 正常；若异常说明哪一格 |
| 部门表没有本周列 | 告诉 Agent「继续」— 应自动插入列 |

更多故障排查见 [SKILL.md](SKILL.md) 文末表格。
